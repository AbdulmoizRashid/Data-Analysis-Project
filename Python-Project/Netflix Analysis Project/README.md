# Netflix Movies Analysis Project

## Overview
This project analyzes a dataset of Netflix movies to extract insights about movie popularity, genre trends, and release patterns. The goal is to understand which movies and genres stand out and how the number of releases has changed over time.

## Tools
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Insights
1. **Drama** dominates as the most frequently released genre on Netflix.  
2. Movies with the **highest average votes** tend to attract more attention and engagement.  
3. **Spider-Man: No Way Home** is the most popular movie based on the metrics analyzed.  
4. **The United States vs. Billie Holiday** ranks as the least popular movie.  
5. The **number of movies released per year** has grown steadily, with a sharp increase after 2000, showing a clear trend of expansion in Netflix content.
